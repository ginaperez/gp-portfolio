Furniture

Built in Visual Studio Code using React, Redux, Express, NodeJS, BCrypt, PostgreSQL, HTML5, and Sass. This full CRUD web app was built with React as an e-commerce furniture store. Users of the site are able to create an account, browse and search the inventory, add items to the cart, modify cart quantities, and check out. View the full site and Github repository.

Void

Built in Visual Studio Code using React, Redux, NodeJS, Express, PostgreSQL, HTML, Sass, and Sockets. This full CRUD web app Void is a group productivity app designed to integrate all parts of working as a group. Once logged in, users are able to join multiple groups, each with its own unique dashboard of tasks. Tasks in each column can be created, edited, and dragged and dropped between each column. A sidebar on the left allows users to view other members of the group with the ability to hide the the sidebar to save space. Using sockets, this app also has live chat functionality between group members. View the full site and Github repository.
